Algorithms for Safety and Inclusivity

1. Dijkstra's Algorithm

Objective: Optimize the shortest paths between public spaces to ensure easy access, especially for vulnerable populations such as elderly, disabled, and women. This helps prioritize routes that are safe, efficient, and inclusive.

  • Input: The distance matrix representing the city’s road network.
  • Output: Shortest paths from residential zones to public service areas, hospitals, and commercial zones.

2. Kruskal’s Algorithm

Objective: Build a Minimum Spanning Tree (MST) that connects all important public spaces with minimal cost and ensures no isolated areas.

  • Input: The city’s distance matrix (edges).
  • Output: MST connecting all public spaces to improve road network connectivity and accessibility.

3. Prim’s Algorithm

Objective: Ensure optimal connectivity between important nodes in the city, with a focus on making public spaces accessible to all residents.

4. Citizen Feedback Portal

Objective: Collect real-time feedback from citizens about the safety and accessibility of public spaces, allowing for continuous improvement in urban planning.

  • Input: Data from citizens reporting issues related to public space safety, barriers for people with disabilities, and general accessibility.
  • Output: Improvements to infrastructure and public space designs based on real-time citizen input.

The Kurakals Algorithm in Citizen Feedback Portal

The Kurakals Algorithm can be applied in a Citizen Feedback Portal in the following ways:

  • Sentiment Analysis: It analyzes the tone of feedback (positive, neutral, or negative) to gauge public sentiment on services.
  • Categorization: The algorithm classifies feedback into relevant categories (e.g., healthcare, transportation) using keywords and patterns.
  • Trend Detection: It identifies emerging issues by tracking frequent concerns across feedback submissions.
  • Prioritization: Based on volume and severity, the algorithm prioritizes urgent issues for immediate attention.
  • Actionable Insights: Aggregated data provides actionable insights, helping governments make informed decisions, allocate resources, and improve services efficiently.

This helps streamline feedback processing, ensuring timely responses and effective governance.

#include 
using namespace std;

class Kruskal {
public:
    struct Edge {
        int u, v, w;
    };

    int n, m, count = 0;
    vector edges, ans;
    int* arr;

    Kruskal() {
        cout << "Enter number of vertices: ";
        cin >> n;
        cout << "Enter number of edges in the graph:";
        cin >> m;

        edges.resize(m);
        cout << "Enter values for u, v, w one by one:\n";
        for (int i = 0; i < m; i++) {
            cin >> edges[i].u >> edges[i].v >> edges[i].w;
        }

        mergeSort(edges);

        arr = (int*)malloc(sizeof(int) * n);
        for (int i = 0; i < n; i++) {
            arr[i] = i;
        }
    }

    void merge(vector& left, vector& right, vector& result) {
        int i = 0, j = 0, k = 0, p = left.size(), q = right.size();
        while (i < p && j < q) {
            if (left[i].w <= right[j].w) {
                result[k++] = left[i++];
            } else {
                result[k++] = right[j++];
            }
        }
        while (i < p) result[k++] = left[i++];
        while (j < q) result[k++] = right[j++];
    }

    void mergeSort(vector& edges) {
        int n = edges.size();
        if (n > 1) {
            vector left(edges.begin(), edges.begin() + n / 2);
            vector right(edges.begin() + n / 2, edges.end());
            mergeSort(left);
            mergeSort(right);
            merge(left, right, edges);
        }
    }

    void Union(int u, int v) {
        int temp = arr[u];
        for (int i = 0; i < n; i++) {
            if (arr[i] == temp)
                arr[i] = arr[v];
        }
    }

    bool Find(int u, int v) {
        if (arr[u] == arr[v])
        return 1;
        else
            return 0;
    }

    void spanning_tree() {
        int i = 0;
        while (count< n - 1 && i < m) {
            if (Find(edges[i].u, edges[i].v)==0) {
                Union(edges[i].u, edges[i].v);
                ans.push_back(edges[i]);
                count++;
            }
            i++;
        }
    }
};

int main() {
    Kruskal k;
    k.spanning_tree();

    cout << "Edges in the Minimum Spanning Tree:\n";
    for (size_t i = 0; i < k.ans.size(); i++) {
    cout << k.ans[i].u << " " << k.ans[i].v << " " << k.ans[i].w << endl;
}

    return 0;
}
