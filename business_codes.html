Algorithms Used:

  • Merge Sort: Sorting algorithm for road data. Application: Sorting road network data for better decision-making.
  • Quick Sort: Sorting algorithm for route prioritization. Application: Used to sort route options for optimization.
  • Dijkstra's Algorithm: Computes the shortest path to minimize travel costs. Application: Used to find the shortest path between critical locations.
